## How to get direct image URLs from Wikimedia Commons

1. Open the file page (e.g. `https://commons.wikimedia.org/wiki/File:Player_Name.jpg`).
2. Click “More details” or the image.
3. Right-click the image → “Copy image address” or use the “Original file” link.
4. Or use: `https://upload.wikimedia.org/wikipedia/commons/` + the path shown on the file page.

## Where to save images in your project

Place downloaded images in:

```
src/assets/players/England/    (create if needed)
src/assets/players/Italy/     (already exists)
src/assets/players/Nigeria/    (already exists – you have JayJay_Okocha.jpg)
```

**Naming suggestion:** `Player_Name.jpg` or `Player_Name.webp` (e.g. `Harry_Kane.jpg`, `Andrea_Pirlo.jpg`).
